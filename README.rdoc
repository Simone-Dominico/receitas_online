# Livro de Receitas Online
Produzido na aula de Programação para Internet 4,
Avaliação dos conhecimentos de Ruby on Rails.
Descrição da Avaliação de Programação para Internet 4
Aluno: Simone Dominico
Ideia: Livro de receitas online
Atores: Administrador, Usuário.
O sistema consiste em um site onde as pessoas poderão se cadastrar para registrar suas receitas e
disponibiliza-las para outros usuários do site. Através de cadastro os usuários poderão armazenar
suas receitas online, e receber as atualizações realizadas no site.
Funcionalidades:
• Cadastro de usuário no site, realizado pelo próprio usuário;
• Os usuários receberão por email as novas receitas cadastradas;
• Login para ter acesso ao cadastro de receitas;
• Receitas poderão conter fotos, upload;
• Área administrativa para acompanhar os participantes da pagina, ver a quantidade de receitas cadastradas;
• As receitas separadas por categorias:
◦ Sobremesas;
◦ Almoço/jantar;
◦ Lanches;
